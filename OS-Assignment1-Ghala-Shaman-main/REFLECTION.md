**Your Answer:**

[Give specific examples from real applications you use (web browsers, games, mobile apps, etc.). Explain why threads are useful in those scenarios. Connect to what you learned in this assignment.]

my answer

Multithreading is widely used in many real-world applications to improve performance and responsiveness. For example, web servers often handle multiple client requests simultaneously using threads so that many users can access the service at the same time. Another example is modern applications such as media players or games where different tasks run concurrently, such as audio playback, graphics rendering, and user input processing. Multithreading allows these tasks to run smoothly without blocking each other. Operating systems also use scheduling algorithms like Round-Robin to manage multiple running programs.
